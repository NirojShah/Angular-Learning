# Pipes in Angular

## What is Pipe?

A Pipe in Angular is used to transform data inside the template without changing the original value.

It takes input data, processes it, and returns transformed output.

Think of it like a formatter.

Example:

```html
<p>{{ name | uppercase }}</p>
```

If:

```ts
name = "niroj";
```

Output:

```text
NIROJ
```

Flow:

```text
Original Data → Pipe → Transformed Data → Display
```

---

## Why do we use Pipes?

Pipes help in formatting data directly in HTML.

Benefits:

* Keeps template clean
* Reusable
* Easy formatting
* Original data remains unchanged
* Improves readability

Example:

Without pipe:

```ts
name = name.toUpperCase();
```

With pipe:

```html
<p>{{ name | uppercase }}</p>
```

---

## Syntax

Basic syntax:

```html
{{ value | pipeName }}
```

With parameter:

```html
{{ value | pipeName:param }}
```

Multiple parameters:

```html
{{ amount | currency:'INR':'symbol' }}
```

Pipe chaining:

```html
{{ name | uppercase | slice:0:4 }}
```

# Built-in Pipes

Angular provides built-in pipes by default.

---

## 1. UpperCasePipe

Converts text into uppercase.

```html
{{ name | uppercase }}
```

Input:

```ts
name = "angular";
```

Output:

```text
ANGULAR
```

---

## 2. LowerCasePipe

Converts text into lowercase.

```html
{{ name | lowercase }}
```

Input:

```ts
name = "ANGULAR";
```

Output:

```text
angular
```

---

## 3. TitleCasePipe

Capitalizes the first letter of each word.

```html
{{ name | titlecase }}
```

Input:

```ts
name = "hello angular";
```

Output:

```text
Hello Angular
```

---

## 4. DatePipe

Formats date values.

```html

```

Input:

```ts
today = new Date();
```

Output:

```text
21-06-2026
```

---

## 5. CurrencyPipe

Formats number as currency.

```html
{{ amount | currency }}
```

Input:

```ts
amount = 2500;
```

Output:

```text
₹2,500.00
```

Custom:

```html
{{ amount | currency:'USD' }}
```

Output:

```text
$2,500.00
```

---

## 6. DecimalPipe

Formats decimal values.

```html
{{ price | number:'1.2-2' }}
```

Input:

```ts
price = 1250.5;
```

Output:

```text
1,250.50
```

---

## 7. PercentPipe

Formats numbers into percentage.

```html
{{ marks | percent }}
```

Input:

```ts
marks = 0.85;
```

Output:

```text
85%
```

---

## 8. SlicePipe

Extracts part of a string or array.

String:

```html
{{ name | slice:0:4 }}
```

Input:

```ts
name = "Angular";
```

Output:

```text
Angu
```

Array:

```html
{{ users | slice:1:3 }}
```

---

## 9. JsonPipe

Converts object into JSON string.

```html
{{ user | json }}
```

Input:

```ts
user = {name:'Niroj', age:25}
```

Output:

```json
{
  "name": "Niroj",
  "age": 25
}
```

Used for debugging.

---

## 10. AsyncPipe

Works with Observables and Promises.

```html
{{ users$ | async }}
```

Features:

* Auto subscribes
* Auto unsubscribes

Very important for API handling.

---

## 11. KeyValuePipe

Converts object into key-value pairs.

Example:

```html
<div *ngFor="let item of user | keyvalue">
  {{ item.key }} : {{ item.value }}
</div>
```

Input:

```ts
user = {
  name: "Niroj",
  age: 25
}
```

Output:

```text
name : Niroj
age : 25
```

---

## 12. I18nPluralPipe

Used for pluralization.

Example:

```html
{{ count | i18nPlural: messageMapping }}
```

TypeScript:

```ts
messageMapping = {
  '=0': 'No messages',
  '=1': 'One message',
  'other': '# messages'
}
```

Output depends on count.

---

## 13. I18nSelectPipe

Used for selecting values based on input.

Example:

```html
{{ gender | i18nSelect: inviteMap }}
```

TypeScript:

```ts
inviteMap = {
  male: 'Invite him',
  female: 'Invite her',
  other: 'Invite them'
}
```

# Custom Pipe

If built-in pipes are not enough, create custom pipes.

Generate:

```bash
ng g p reverse
```

Angular creates:

```text
reverse.pipe.ts
```

---

## Example: Reverse Pipe

### Pipe File

```ts
import { Pipe, PipeTransform } from '@angular/core';

@Pipe({
  name: 'reverse'
})
export class ReversePipe implements PipeTransform {
  transform(value: string): string {
    return value.split('').reverse().join('');
  }
}
```

---

### HTML

```html
<p>{{ name | reverse }}</p>
```

Input:

```ts
name = "Angular";
```

Output:

```text
ralugnA
```

# Pure Pipe vs Impure Pipe

## Pure Pipe

Runs only when input reference changes.

Default behavior.

Example:

```ts
@Pipe({
  name: 'myPipe',
  pure: true
})
```

Advantages:

* Faster
* Better performance

---

## Impure Pipe

Runs on every change detection.

Example:

```ts
@Pipe({
  name: 'myPipe',
  pure: false
})
```

Disadvantages:

* Slower
* Performance heavy

Use carefully.

---

# Chaining Pipes

Multiple pipes can be used together.

Example:

```html
{{ name | uppercase | slice:0:4 }}
```

Input:

```ts
name = "angular";
```

Output:

```text
ANGU
```

Flow:

```text
angular → uppercase → ANGULAR → slice → ANGU
```

# Common Mistakes

## Using pipes inside TypeScript

Wrong:

```ts
name | uppercase
```

Correct:
Use pipes in templates only.

---

## Forgetting pipe symbol

Wrong:

```html
{{ name uppercase }}
```

Correct:

```html
{{ name | uppercase }}
```

---

## Using impure pipes unnecessarily

Can affect performance.
