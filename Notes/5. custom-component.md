# Custom Component

A **Custom Component** is the basic building block of an Angular Application.  
Here we will see how to create a component manually **without using Angular CLI**.

## Following is the guideline to create a Custom Component:

- First create a folder inside the `app` folder.  
  **Example:** `signup`

- Inside the `signup` folder create a TypeScript file.  
  **Example:** `signup.ts`

Folder Structure:

```txt
app/
 ├── signup/
 │   ├── signup.ts
```

---

## Step 1: Create the Component File

Inside `signup.ts`, write the component code:

```ts
import { Component } from '@angular/core';

@Component({
  selector: 'app-signup',
  templateUrl: './signup.html',
  styleUrls: ['./signup.css']
})
export class SignupComponent {
  title = "Signup Page";
}
```

Here:

- `selector` → Used to call the component
- `templateUrl` → External HTML file
- `styleUrls` → External CSS file

---

## Step 2: Create HTML File

Inside the same folder create:

```txt
signup.html
```

Write:

```html
<h1>{{ title }}</h1>
<p>Welcome to Signup Page</p>
```

---

## Step 3: Create CSS File

Inside the same folder create:

```txt
signup.css
```

Write:

```css
h1 {
  color: blue;
}

p {
  font-size: 18px;
}
```

---

## Final Folder Structure

```txt
app/
 ├── signup/
 │   ├── signup.ts
 │   ├── signup.html
 │   ├── signup.css
```

---

## Using the Component

To use the component, call it inside another component:

Example:

```html
<app-signup></app-signup>
```

Angular will render the signup component.

---

# Using Inline Template and Styles

If you do not want to create separate HTML and CSS files, you can define them directly inside the component using:

- `template`
- `styles`

Example:

```ts
import { Component } from '@angular/core';

@Component({
  selector: 'app-signup',
  template: `
    <h1>{{ title }}</h1>
    <p>Welcome to Signup Page</p>
  `,
  styles: [`
    h1 {
      color: red;
    }

    p {
      font-size: 18px;
    }
  `]
})
export class SignupComponent {
  title = "Signup Page";
}
```

Here:

- `template` → Inline HTML
- `styles` → Inline CSS

No need for:

```txt
signup.html
signup.css
```

---

## Difference Between External and Inline

| External Files | Inline |
|---|---|
| Uses `templateUrl` | Uses `template` |
| Uses `styleUrls` | Uses `styles` |
| Better for large code | Better for small code |
| Easy to maintain | Quick to write |

---

## Quick Summary

To create a custom component manually:

1. Create a folder
2. Create `.ts` file
3. Add `@Component()` decorator
4. Define selector
5. Add HTML using `templateUrl` or `template`
6. Add CSS using `styleUrls` or `styles`
7. Export the class
8. Use selector inside HTML

Example:

```html
<app-signup></app-signup>
```

Output:

```txt
Signup Page
Welcome to Signup Page
```
