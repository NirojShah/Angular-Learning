# Property Binding in Angular

## 1. What is Property Binding?

**Property Binding** is a one-way data binding technique in Angular.

It is used to bind data from the component (TypeScript) to the HTML element properties.

It means Angular sends data from the class to the view.

Syntax:

```html
[property]="value"
```

Simple meaning:

Property Binding is used to dynamically set values for HTML properties.

Examples:

- `src`
- `href`
- `disabled`
- `value`
- `id`

---

## 2. Why use Property Binding?

We use Property Binding when we want to:

- Set element properties dynamically
- Update values automatically when data changes
- Control UI behavior using TypeScript variables

Example:

Instead of writing fixed values:

```html
<button disabled="true">Submit</button>
```

Use dynamic values:

```html
<button [disabled]="isDisabled">Submit</button>
```

This makes the application flexible.

---

## 3. Basic Syntax

Property Binding uses square brackets `[]`

Syntax:

```html
[element-property]="component-variable"
```

Example:

```html
<img [src]="imageUrl">
```

Here:

- `src` = HTML property
- `imageUrl` = Component variable

---

## 4. Simple Example

### app.component.ts

```ts
import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html'
})
export class AppComponent {
  name = "Angular";
  isDisabled = false;
  imageUrl = "https://example.com/logo.png";
}
```

---

### app.component.html

```html
<h1 [title]="name">Welcome</h1>

<button [disabled]="isDisabled">
  Click Me
</button>

<img [src]="imageUrl" alt="Logo">
```

Explanation:

- `[title]` sets tooltip
- `[disabled]` controls button state
- `[src]` loads image dynamically

---

## 5. Class and Style Binding

Property Binding can also be used for classes and styles.

### Class Binding

Used to add/remove CSS classes.

Example:

```html
<p [class.active]="isActive">
  Hello Angular
</p>
```

Component:

```ts
isActive = true;
```

If `true`, class will be added.

---

### Style Binding

Used to apply styles dynamically.

Example:

```html
<p [style.color]="textColor">
  Welcome
</p>
```

Component:

```ts
textColor = "blue";
```

Output:

Text color becomes blue.

---

## 6. Property Binding with Signals

Property Binding works directly with Signals.

Example:

### app.component.ts

```ts
import { Component, signal } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html'
})
export class AppComponent {
  isDisabled = signal(false);
  buttonColor = signal('green');
}
```

---

### app.component.html

```html
<button [disabled]="isDisabled()">
  Submit
</button>

<p [style.color]="buttonColor()">
  Signal Example
</p>
```

Here:

Signals automatically update UI when values change.

---

## 7. Do's and Don'ts

### Do's

✔ Use property binding for dynamic values

Example:

```html
<img [src]="profileImage">
```

✔ Use for boolean properties

Example:

```html
<button [disabled]="isLoading">
```

✔ Use signals for reactive updates

Example:

```html
<div [hidden]="isHidden()">
```

---

### Don'ts

❌ Do not use interpolation inside property binding

Wrong:

```html
<img [src]="{{imageUrl}}">
```

Correct:

```html
<img [src]="imageUrl">
```

---

❌ Do not use property binding for static values

Wrong:

```html
<button [disabled]="true">
```

Better:

```html
<button disabled>
```

## 8. Interview Questions

### Q1. What is Property Binding?

**Answer:**

Property Binding is a one-way binding used to bind data from component to HTML properties.

---

### Q2. What is the syntax of Property Binding?

**Answer:**

```html
[property]="value"
```

---

### Q3. Difference between Interpolation and Property Binding?

**Answer:**

| Interpolation | Property Binding |
|---|---|
| Uses `{{ }}` | Uses `[ ]` |
| Displays data | Sets element properties |

Example:

Interpolation:

```html
<h1>{{ name }}</h1>
```

Property Binding:

```html
<img [src]="imageUrl">
```

---

### Q4. Can we use Property Binding with Signals?

**Answer:**

Yes.

Example:

```html
<button [disabled]="isDisabled()">
```

---

### Q5. What is Class Binding?

**Answer:**

Used to dynamically add/remove classes.

Example:

```html
<p [class.active]="isActive"></p>
```

---

### Q6. What is Style Binding?

**Answer:**

Used to apply dynamic styles.

Example:

```html
<p [style.color]="color"></p>
```
